This project focuses on predicting employee promotions using a binary classification approach within the Data Mining domain.
Starting from raw HR data, I conducted a complete data preprocessing pipeline including data cleaning, handling missing values, outlier detection, and feature engineering (encoding, transformation, and scaling).I performed exploratory data analysis (EDA), including distribution analysis, correlation study, and detection of skewness and multicollinearity.
The dataset was then partitioned into training and testing sets, and multiple models were implemented, including Gaussian Naive Bayes, Random Forest, and Neural Networks.
Finally, I evaluated and compared model performance to identify the most effective approach for predicting employee promotions.
